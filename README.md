## React Native Login & Register App

Aplikasi mobile sederhana menggunakan **React Native (Expo)** yang memiliki fitur:
- Login
- Registrasi akun
- Validasi form menggunakan Formik & Yup
- Upload foto profil (Image Picker)
- UI modern dan responsif

## Fitur Utama
- Login
- Input email & password
- Validasi form
- Error handling jika login salah

## Register
- Input:
  - Nama lengkap
  - Email
  - Nomor HP
  - Password & Konfirmasi Password
- Validasi lengkap (Yup)
- Upload foto profil dari galeri/kamera
- Notifikasi berhasil registrasi
